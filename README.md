# Deepa Tilwani Research Portfolio

Static research portfolio with a Scholar-backed publications feed.

## Data flow
- `assets/scholar-publications.json` and `assets/manual-publications.json` feed the site data bundle.
- `assets/scholar-metrics.json` stores Scholar profile metrics such as citations and h-index.
- `assets/site-data.js` is the file the pages actually render from.
- `scripts/sync_scholar.py` can refresh the Scholar publication JSON.
